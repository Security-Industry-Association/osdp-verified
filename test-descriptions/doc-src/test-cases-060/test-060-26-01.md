## Test 060-26-01 File Transfer ##

### Description ###

Command for PD to transfer a block of data, called a "file".

#### PD ####

PD processes a FILETRANSFER command and receives a "file" including
control details and status.

#### ACU ####

ACU sends FILETRANSFER command and transmits "file", using one or more
osdp_FILETRANSFER commands and processing osdp_FTSTAT responses.

### Purpose ###

Support firmware updates of PDs, configuration changes and other
applications.

### Test action ###

#### PD ####

Send a test file, confirm it either processes the file properly if it is 
appropriate for that PD, or, confirm it reacts appropriately (via osdp_FTSTAT) if
sent a test file.

#### ACU ####

Send a file.
