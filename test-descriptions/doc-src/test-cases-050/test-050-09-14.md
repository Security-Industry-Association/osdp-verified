## Test 050-09-14 Rogue SCS Poll ##

### Description ###

If a PD receives a packet with an SCS header while it
is operating unencrypted (no secure channel active) then
it must reject the packet with a NAK, encryption required.  It must not
use an SCS header in the response as there is no secure channel active.

#### PD ####

Determine the device supports RS-485 communications at designated
speed(s) and address(es).

#### ACU ####

t.b.d.

### Purpose ###

t.b.d.
