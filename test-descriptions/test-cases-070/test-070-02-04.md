## Test 070-02-04 ISTATR Response to Poll (Supervised) ##

### Description ###

This is for supervised input devices.
Determine the ISTATR response matches PDCAP response.

#### PD ####

Test that the PD responds with ISTATR to poll commands.

#### ACU ###

Accept this response, react benignly if not processed.

### Purpose ###

Determine ability to support unsolicited input status responses.

### Test Action ###

#### PD Test ####

Set up the PD with no wires connected to the inputs.
Confirm the ISTATR response shows an appropriate value for disconnected
connections i.e. all 1's.

#### ACU Test ####

Observe response is processed or benign.
