## Test 070-07-02 Supervised ISTATR Input Status Report ##

### Description ###

#### PD ####

Provide change in input status in 0x49 response to osdp_ISTAT 0x65
command if there is a change in input status monitored by the PD.
This test specifically exercises supervised inputs.

#### ACU ####

Accept this capability, and whether it is used.

### Purpose ###

Determine ability to monitor status and changes of status of inputs of
the PD. Confirm PD can detect change in supervised state.

### Test action ###

#### PD Test ####

Configure the PD with no input signalls attached.

Determine the input status capabilities reported and test input
capabilities.  For this test must include at least one supervised 
input.

The istatr should
return an unsupervised value, typically all 1's.

Support capabilities reported
